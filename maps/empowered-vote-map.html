How this map works

A self-contained interactive mind map. One HTML file, no accounts, no subscription. Converted from a MindMeister export with structure, colors, links, and images intact.

Controls

  • Drag the background to pan, scroll or pinch to zoom.
  • The round +/− buttons on a node expand or collapse its branch. A number shows how many topics are hidden inside.
  • Click any node to open its links, videos, and images in the detail card. ↗ marks a link, ▶ a video, 📎 an attached image.
  • Type in the search box to find topics anywhere in the map; matching branches expand automatically.
  • Hover a node and drag the slim bar on its outer edge to make the box wider or narrower. Widths are remembered in this browser.
  • Expand all opens every branch; Collapse returns to the overview.

A note on links

Opened locally or from a normal web host, every link and video opens in a new tab. Viewed as a claude.ai artifact, the security sandbox blocks the page from opening other sites; use the copyable address shown under each link button instead.

Editing

All content lives in one MAP object at the top of this file's script. Edit the object, save, and the layout rebuilds itself. To restructure it conversationally, hand the file back to Claude.
