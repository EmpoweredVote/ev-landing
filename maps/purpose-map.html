How this map works

A self-contained interactive mind map of Empowered Vote's purpose. One HTML file, no accounts, no subscription.

Controls

  • Drag the background to pan, scroll or pinch to zoom.
  • Nodes with a soft glowing ring have an explainer page. Click one to open it, complete with a little animated guide.
  • The round +/− buttons on a node expand or collapse its branch. A number shows how many topics are hidden inside.
  • Type in the search box to find topics anywhere in the map; matching branches expand automatically.
  • Hover a node and drag the slim bar on its outer edge to change its width. Widths are remembered in this browser.
  • Expand all opens every branch; Collapse returns to the overview.

A note on links

Opened locally or from a normal web host, every link and video opens in a new tab. Viewed inside a sandboxed preview, use the copyable address shown under each link instead.

Editing

All content, including every explainer, lives in one MAP object at the top of this file's script. Edit it, save, and the layout rebuilds itself. To restructure it conversationally, hand the file back to Claude. Nothing here depends on MindMeister.
